DriveCatalogue V5
=================

Quick Start
-----------
1. Install Python 3.13 on Windows.
2. Install requirements: `pip install -r requirements.txt`.
3. Run the app: `python DriveCatalogue_V5.py`.

Build EXE
---------
1. Run `build.bat` which creates a venv and packages with PyInstaller.
2. The resulting EXE is in the `dist` folder as `DriveCatalogue_V5.exe`.

Usage
-----
* Dashboard shows summary stats.
* Search tab queries files by name, size and extension.
* Reports tab displays used vs free space per root.
* Queue tab adds paths with aliases and scans them sequentially.
* Settings tab configures options and scheduling.
